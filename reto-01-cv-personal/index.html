Acerca de mi

Soy Data Engineering, Data Analysis y Developer, con más de 25 años de experiencia en diversas áreas de TI, incluyendo desarrollo de sistemas, bases de datos. Mi trayectoria me ha permitido consolidar una profunda experiencia en el ciclo de vida completo de los datos, desde la integración hasta la visualización.

Poseo experiencia en SQL y una sólida habilidad en procesos ETL, destacando mi trabajo con SQL Server Integration Services (SSIS) y Data Factory. Actualmente, trabajo como Data Engineer, trabajando con Snowflake, Azure SQL Azure Data Factory, Azure Functions y Power BI.

Mi perfil técnico se complementa con experiencia en programación Python (Jupyter, Pandas, Matplotlib, Seaborn), así como en .Net, HTML, CSS, JavaScript y MVC. Tengo conocimientos básicos en la nube de Microsoft Azure (SQL Azure, Azure Data Factory, Azure Functions y Data Fabric).

Además de mis habilidades técnicas, cuento con capacidad en consultoría y capacitación, con experiencia en el contacto directo con clientes nacionales e internacionales. Mi nivel de inglés es B1 intermedio.

Apasionado por crear soluciones de negocio sencillas y útiles, siempre buscando la eficiencia y la mejora constante en los procesos de integración y análisis de datos.
